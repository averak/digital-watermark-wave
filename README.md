# digital-watermaker

![Python 3.6](https://img.shields.io/badge/python-3.9-green.svg)
[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

This app is digital watermaker for wave.

## Requirement

- Python 3.9
- pipenv

## Develop

### Installation

```bash
$ pipenv install
```

### Lint & Format

```bash
$ pipenv run lint
$ pipenv run format
```
